# Decisions — 决策日志

> **分轻重两档写**（见 `memory/AGENT_PROTOCOL.md` §2.1）：
>
> 【轻量单行】次级偏好 / 备忘 / 非取舍类——直接一行到"## 备注（轻量）"段：
> ```
> - YYYY-MM-DDTHH:MM:SSZ：一句话说清楚是什么决定
> ```
> 脚本：`scripts/memory/note "msg"`
>
> 【完整五段式】真·取舍（有 ≥2 个被认真考虑又被否掉的备选才用）：
> ```
> - 时间：YYYY-MM-DDTHH:MM:SSZ (UTC)
>   - 背景：
>   - 决策：
>   - 备选：
>   - 理由：
>   - 影响：
> ```
> 脚本：`scripts/memory/note "msg" --decision`
>
> **合并规则**：一次会话里的多个相关决策合并为一个时间戳块，不要拆成 N 条同时间戳条目。
> 完整历史记录：`memory/archive/`
